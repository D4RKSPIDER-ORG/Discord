# Discord Quest Completer

A simple way to complete Discord quests using the browser console.

---

# How to Use

## Step 1 — Enable Developer Mode

1. Open **Discord Settings**.
2. Go to **Advanced**.
3. Turn **Developer Mode** **ON**.

Path:  
**Settings → Advanced → Developer Mode → Enable**

---

## Step 2 — Open a Quest

1. Go to the **Discord Quest tab**.
2. Open the **Quest Area**.
3. Select a quest.
4. Accept the quest.
5. Make sure you select the **PC quest** (this only works on Discord PC).

---

## Step 3 — Open Console

1. Press **CTRL + SHIFT + I**.
2. The **Developer Tools** window will open.
3. Click the **Console** tab at the top.

---

## Step 4 — Run the Script

1. Paste the **script** into the console.
2. Press **Enter**.

The script will automatically complete **all quests you have already accepted or started**.
